# 移动零

难度：简单

## 题目

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**要求**：必须在原数组上操作，不能使用额外的数组空间。

## 示例

```
输入：nums = [0, 1, 0, 3, 12]
输出：[1, 3, 12, 0, 0]
```

```
输入：nums = [0]
输出：[0]
```

## 解法

### 额外数组 O(n)

创建新数组存储非零元素，再填充 0。但不符合题目"原地操作"的要求。

### 双指针 - 覆盖版 O(n)

两次遍历：第一次把非零元素移到前面，第二次把后面填 0。

```python
def moveZeroes(self, nums: List[int]) -> None:
    i = 0
    # 第一遍：把所有非零元素移到前面
    for j in range(len(nums)):
        if nums[j] != 0:
            nums[i] = nums[j]
            i += 1
    # 第二遍：把后面的位置都填0
    while i < len(nums):
        nums[i] = 0
        i += 1
```

### 双指针 - 交换版 O(n) ✓

一次遍历完成。核心思路：
- **慢指针 i**：指向"下一个非零元素应该放的位置"
- **快指针 j**：遍历整个数组
- 当 j 找到非零元素时，与 i 位置交换，然后 i 前进

```python
def moveZeroes(self, nums: List[int]) -> None:
    i = 0
    for j in range(len(nums)):
        if nums[j] != 0:
            nums[i], nums[j] = nums[j], nums[i]
            i += 1
```

**为什么交换有效？**
- 当 `i == j` 时，交换自己，不影响结果
- 当 `i < j` 时，i 指向的一定是 0（因为非零元素都被前移了），把 j 的非零元素换过来

## 复杂度

- 时间：O(n)，只需一次遍历
- 空间：O(1)，原地操作

## 要点

1. **快慢指针**：i 慢（标记位置），j 快（寻找元素）
2. **元组解包**：Python 支持 `a, b = b, a` 语法直接交换变量
   ```python
   # 交换原理
   nums[i], nums[j] = nums[j], nums[i]
   # 等价于：
   temp = nums[i]
   nums[i] = nums[j]
   nums[j] = temp
   ```
3. 一次遍历即可完成，无需后续填零操作

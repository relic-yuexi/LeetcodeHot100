# 双指针

## 核心思想

利用两个指针在数组或链表中移动，通过指针的协同工作降低时间复杂度，通常可以将 O(n²) 优化到 O(n)。

## Python 中的常用操作

| 操作 | 用途 | 复杂度 |
|------|------|--------|
| `left, right = 0, len(arr)-1` | 初始化对撞指针 | O(1) |
| `while left < right` | 对撞指针循环条件 | - |
| `arr.reverse()` | 原地反转数组 | O(n) |
| `arr[::-1]` | 返回反转后的新数组 | O(n) |

## 常见套路

1. **对撞指针**：左右两端向中间移动，常用于有序数组
   - 两数之和（有序数组）
   - 三数之和
   - 盛最多水的容器

2. **快慢指针**：两个指针同向移动，速度不同
   - 删除重复元素
   - 移动零
   - 链表环检测

3. **固定间距指针**：两指针保持固定距离
   - 删除链表倒数第N个节点

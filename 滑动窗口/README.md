# 滑动窗口

## 核心思想

用左右两个指针维护一个窗口，右指针不断扩张，左指针在条件不满足时收缩，使得每个元素最多被访问两次，将暴力的 O(n²) 优化为 O(n)。

## Python 中的常用操作

| 操作 | 说明 | 适用场景 |
|------|------|------|
| `dict[char] = index` | 记录字符最后出现的位置 | 需要跳跃式移动左指针 |
| `set.add() / set.discard()` | 维护窗口内元素集合 | 需要逐步移动左指针 |
| `collections.Counter()` | 统计窗口内字符频次 | 字母异位词、最小覆盖子串 |
| `collections.defaultdict(int)` | 默认值为 0 的频次字典 | 避免 KeyError |

## 常见套路

1. **定长窗口**：窗口大小固定，右指针每进一步左指针也进一步
   - 字母异位词匹配
   - 固定大小子数组的最大和

2. **变长窗口（求最长）**：右指针扩张，条件不满足时左指针收缩
   - 无重复字符的最长子串
   - 关键：`left = max(left, dict[char] + 1)` 防止左指针回退

3. **变长窗口（求最短）**：右指针扩张直到满足条件，然后左指针收缩尝试更短的答案
   - 最小覆盖子串

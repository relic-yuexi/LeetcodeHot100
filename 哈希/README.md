# 哈希表

## 核心思想

用空间换时间，将 O(n) 的查找降为 O(1)。

## Python 中的哈希结构

| 结构 | 用途 | 操作复杂度 |
|------|------|-----------|
| `dict` | 存储键值对 | 查/增/删 O(1) |
| `set` | 存储不重复元素 | 查/增/删 O(1) |
| `collections.Counter` | 计数 | 统计频率 |
| `collections.defaultdict` | 带默认值的字典 | 避免 KeyError |

## 常见套路

1. **两数之和类**：边遍历边存，查找 `target - num`
2. **计数统计类**：用 Counter 或 dict 统计频率
3. **去重/判重类**：用 set 记录已访问元素
4. **分组类**：用 dict 按某种规则分组

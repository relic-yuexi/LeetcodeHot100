# 字母异位词分组

难度：中等

## 题目

给定一个字符串数组 `strs`，将字母异位词组合在一起，可以按任意顺序返回。

字母异位词：由相同字母重新排列得到的字符串，字母种类和数量完全相同。

## 示例

```
输入：strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出：[["bat"], ["nat", "tan"], ["ate", "eat", "tea"]]
```

## 解法

### 排序 + 哈希分组 O(n·k·log k) ✓

核心思路：异位词排序后得到相同的字符串，用排序结果作为 dict 的 key 进行分组。

```python
def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
    d = {}
    for s in strs:
        key = tuple(sorted(s))
        if key in d:
            d[key].append(s)
        else:
            d[key] = [s]
    return list(d.values())
```

### 计数 + 哈希分组 O(n·k) ✓

核心思路：用长度 26 的数组统计每个字母出现次数，用计数结果作为 key，避免排序。

```python
def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
    d = {}
    for s in strs:
        k = [0] * 26
        for c in s:
            k[ord(c) - ord('a')] += 1
        key = tuple(k)
        if key in d:
            d[key].append(s)
        else:
            d[key] = [s]
    return list(d.values())
```

## 复杂度

n = 字符串个数，k = 最长字符串长度

| 解法 | 时间 | 空间 |
|------|------|------|
| 排序分组 | O(n·k·log k) | O(n·k) |
| 计数分组 | O(n·k) | O(n·k) |

## 要点

1. `sorted()` 返回 list，list 不可哈希，必须转 `tuple` 才能做 dict 的 key
2. 分组问题的通用套路：找到能唯一标识每组的 key，用 dict 收集
3. `d.values()` 取所有值，`d.keys()` 取所有键——注意要加括号调用
4. `ord(c)` 获取字符的 ASCII 码，`ord(c) - ord('a')` 将字母映射到 0-25
5. 计数解法省掉了排序的 log k，理论更优
